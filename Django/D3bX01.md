## Cài đặt Django
Lưu ý rằng đối với các hệ điều hành khác nhau (Linux, Windows và Mac) thì câu lệnh sẽ khác dưới một chút xíu, các bạn Google thêm để ra câu lệnh đúng cho máy mình.

{% highlight python %}
# Tạo folder chứa project và chuyển vào folder đó
mkdir Mysite_project && cd Mysite_project

# Dùng Python tạo virtual env
python -m venv env

# Activate 
env/Scripts/activate #Windows 

source env/bin/activate #Linux

source env/Scripts/activate #Bash trong Windows 

# Cài Django
pip install django # Version mới nhất lúc này là 3.1

# Lưu các package đã cài đặt
pip freeze > requiremts.txt 
{% endhighlight %}

## Tạo một Django project

{% highlight python %}
django-admin startproject mysite .
{% endhighlight %}
